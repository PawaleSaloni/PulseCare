# 📘 PulseCare - User Stories

---

## 👤 Patient User Stories

### US_PATIENT_01
As a Patient, I want to register so that I can create my account.

### US_PATIENT_02
As a Patient, I want to login so that I can access my dashboard.

### US_PATIENT_03
As a Patient, I want to view the list of doctors so that I can choose a doctor.

### US_PATIENT_04
As a Patient, I want to search doctors by specialization so that I can find the right doctor.

### US_PATIENT_05
As a Patient, I want to check doctor availability so that I can book an appointment.

### US_PATIENT_06
As a Patient, I want to book an appointment so that I can consult a doctor.

### US_PATIENT_07
As a Patient, I want to view my appointments so that I can track them.

### US_PATIENT_08
As a Patient, I want to cancel an appointment so that I can reschedule later.

### US_PATIENT_09
As a Patient, I want to view my prescriptions so that I can follow treatment.

---

## 👨‍⚕️ Doctor User Stories

### US_DOCTOR_01
As a Doctor, I want to login so that I can access my dashboard.

### US_DOCTOR_02
As a Doctor, I want to manage my availability so that patients can book appointments.

### US_DOCTOR_03
As a Doctor, I want to view my appointments so that I can manage my schedule.

### US_DOCTOR_04
As a Doctor, I want to update appointment status so that I can mark them as completed.

### US_DOCTOR_05
As a Doctor, I want to add prescriptions so that patients can follow treatment.

---

## 👩‍💼 Admin User Stories

### US_ADMIN_01
As an Admin, I want to login so that I can manage the system.

### US_ADMIN_02
As an Admin, I want to add doctors so that patients can book appointments.

### US_ADMIN_03
As an Admin, I want to view all patients so that I can monitor users.

### US_ADMIN_04
As an Admin, I want to view all appointments so that I can manage system activity.

### US_ADMIN_05
As an Admin, I want to manage users so that I can control system access.
